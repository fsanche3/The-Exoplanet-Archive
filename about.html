About this Page

This informational website is intended to give the users a mental model of what a discovered exoplanet can look like. By selecting a planet and interacting with the Tableau dashboard provided, a user is able to compare discovered exoplanet measurements with the planets in our own solar system. Allowing them to interpret individual elements of each newly discovered exoplanet.

The goal of this website is not only to promote my full stack web developing and data visualization ability, but also to increase attention to new exoplanet discoveries made by NASA, including the new James Webb Telescope.
